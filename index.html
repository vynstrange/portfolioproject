Information About me

I've had an avid interest since I first took a VisualBasic course back in midde school. Since then, I've taken whatever I could in school to learn more about programming. I took several Java programming classes in high school, and I extended that interest in my college years while I pursued my degree in Information Technology. There I was exposed to more Java as well as some SQL programming. Since college, my mind has always wandered back to the wonderful world of programming and the vast challenges and opportunities it presented. There are so many different technologies out there to learn, and I'm very passionate about learning. On top of being a portfolio website, I consider this page a passion project of its own accord, testing my own skills to write functional code that really showcases my abilities as well as my desire to continue learning.
